====
nimtetris
====

|nimble-version| |nimble-install| |gh-actions|

A terminal tetris in Nim.

.. contents:: Table of contents

Usage
=====

.. code-block:: shell

   $ nimtetris

Key bindings
------------

Vim like key-bindings.

=============== ===================
Key             Motion
=============== ===================
H               Move left
J               Move down
L               Move right
U               Rotate left
O               Rotate right
<Space>/<Enter> Move down to bottom
=============== ===================

Installation
============

.. code-block:: shell

   $ nimble install -Y nimtetris
